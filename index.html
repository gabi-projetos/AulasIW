Introdução ao HTML

HTML é a linguagem de marcação padrão para criar páginas da Web.

O que é HTML?

  • HTML significa Hyper Text Markup larguagem
  • HTML é a linguagem marcação padrão para a criação de páginas da web
  • HYML descreve a estrutura de um página da web
  • HTML consiste em uma série de elementos
  • Elementos HTML dizem ao navegador como exibir o conteúdo
  • Elementos HTML rotulam peças de conteúdo como "este é um título", "este é um parágrafo", "este é um link", etc.

Como ja vimos anteriormente , as CSS sãõ as Cascading Style Sheets (Folha de Estilo em Cascata). Elas são usadas para configura um resultado visual dos elementos HTML. As configurações dos CSS são realizados através dos seletores. Vamos ver a anatomia de um seletor.

Tipos de CSS
